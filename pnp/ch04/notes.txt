1. Name four major types of error and briefly describe each one 
  - Compile error: syntax errors that occur in the source code that are caught by the compiler 
  - Linker error: in the process of forming an executable; the object files form mismatches causing there to be a linking issue 
  - Type errors: when there is a type mismatch between the object and its value 
  - Run-time error: logic errors which are identifiable only after the program has been executed
