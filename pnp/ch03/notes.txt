Chapter 3: Computation Review

"If it doesn't have to produce correct results, I can make it arbitrarily fast" 


1. What is a computation?
   A computation is an operation that produces a result from inputs.

2. What are inputs and outputs?
   Inputs are values given to a computation.
   Outputs are the results produced from those inputs.

3. What are the three requirements for expressing computations?
   Correctness
   Simplicity
   Efficiency

4. What is an expression?
   An expression is something that produces a value.

5. Statement vs expression
   An expression produces a value.
   A statement performs an action.

6. What is an lvalue?
   An lvalue is an object that has a location in memory and can appear on the left side of an assignment.

7. What is a constant expression?
   An expression evaluated at compile time that does not change.

8. What is a literal?
   A value written directly in the source code.

9. What is a symbolic constant?
   A named constant (using const or constexpr) that does not change.

10. What is a magic constant?
    A raw number in code with no explanation.

11. Operators for integers and floating-point values

* * * /

> < >= <= == !=
> = += -= *= /=

12. Operators only for integers
    % (modulus)

13. Operators for strings

* (concatenation)
  == != < >

14. When to use a switch statement?
    When comparing one variable against fixed constant values.

15. Problems with switch
    Only works with integral types
    Limited flexibility
    Requires break

16. Parts of a for-loop and execution order
    for (init; condition; increment)
    Execution: init → condition → body → increment → repeat

17. for-loop vs while-loop
    for: known number of iterations
    while: condition-based, unknown count

18. What does char foo(int x) mean?
    A function named foo that takes an int and returns a char.

19. When to define a separate function?
    When code is reusable
    To organize logic
    To improve readability

20. What can you do with an int but not a string?
    Arithmetic operations (+ - * / %)

21. What can you do with a string but not an int?
    Get length (.size())
    Concatenate (+)

22. Index of the third element in a vector
    2

23. How to print every element of a vector
    Use a loop (e.g., range-based for loop)

24. What does vector<char> alphabet(26) do?
    Creates a vector of 26 chars initialized to '\0'

25. What does push_back() do?
    Adds an element to the end of the vector

26. What does size() do?
    Returns the number of elements in the vector

27. Why is vector useful?
    Dynamic size
    Easy to use
    Safe
    Efficient

28. How do you sort a vector?
    std::sort(v.begin(), v.end());
