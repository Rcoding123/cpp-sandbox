3.3 Expressions

Expressions can use multiple operators to calculate a certain value 

3.3.1 Constant expressions
What is a constant?
A value that will remain the same throughout a programs life 

How to make something constant?
Use a constexpr symbol -> a value that will be known at compile time (a constant expression)

How about if the value is a constant but initialized with no value? (The value never changes after initialization)
Use a const symbol -> use for variables that do not change value after initialization
